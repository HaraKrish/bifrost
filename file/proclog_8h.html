8.2.13. File proclog.h¶

Defines functions for creating and updating real-time process logs that are stored in /dev/shm/bifrost/<pid>/<logname>

Note
Log files are deleted at process shutdown. Logs left behind by killed processes may be cleaned up during subsequent process launches.
